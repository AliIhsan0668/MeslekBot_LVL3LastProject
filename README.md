# MeslekBot_LVL3LastProject
Proje Kodland platformunda aldığım level3 dersi bitirme projemdir. Bu proje bitirme projesi olduğu ve belli kurallar çerçevesinde yapılan bir proje olduğundan belirli bir seviyededir.
# Kariyer Formu Discord Botu
Bu bot, kullanıcılara sorular sorarak meslek önerisi yapan basit bir Discord uygulamasıdır. Bazı sorular mesajla, bazıları butonla cevaplanır. Verilen tüm cevaplar SQLite veritabanına kaydedilir.
## Özellikler
*kariyer komutu ile form başlatılır.
Yaş bilgisi mesaj ile alınır.
Alan seçimi butonlar ile yapılır.
İlgi alanı mesaj ile alınır.
Teknoloji ilgisi butonlar ile alınır.
Verilen cevaplara göre meslek önerisi oluşturulur.
Cevaplar SQLite veritabanına kaydedilir.
